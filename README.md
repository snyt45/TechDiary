# TechDiary
Sites created with Hugo

## 運用
# ローカルでの記事確認
１． 記事作成
```
hugo new posts/20200627/test_post/index.md
```

２． ビルド
```
hugo
```

３． サーバー起動
```
hugo server
```

４． 生成されたURLにアクセス

# デプロイ
