# TechDiary
Sites created with Hugo

# ローカルでの記事確認
記事作成
```
hugo new posts/20200627/test_post/index.md
```

↓

ビルド
```
hugo
```

↓

サーバー起動
```
hugo server
```

↓

生成されたURLにアクセス
